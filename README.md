To manage you todo list of a day.

Most important it manages your 

           COMPLETED Tasks
           DUE Tasks
           Can DELETE task
           View ALL task
           Theme according to your mood i.e light nad dark.
           

App hosted: https://sanskritikhatiyan.github.io/todo_list/
